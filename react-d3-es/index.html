Qué es D3.js?

  • Data-Driven Documents
  • Librería de visualización de datos de bajo nivel y uso general
  • Manipula documentos basados en datos
  • Estándares Web (SVG, HTML, CSS y Canvas)
  • Permite interacciones con tus gráficas
* El nombre D3 viene de "data-driven documents" * con una API inmensa * y centrada en transformaciones matemáticas de esos mismos datos * Basado en la web: SVG, HTML, CSS y canvas * Una de sus más celebradas características es su integración de animaciones, y cómo permite crear interacciones cautivadoras * Es una caja de herramientas para crear visualizaciones interactivas en la web. * No es una librería de gráficas, pero luego vamos a revisar algunos proyectos basados en D3 que sí lo son

Cómo funciona?

  • Carga datos
  • Combina los datos con elementos del DOM
  • Transforma esos elementos
  • Transiciona entre estados
* D3 carga datos y los incorpora en el DOM * Combina esos datos con elementos del DOM * y transforma esos elementos, * transicionando entre estados si es necesario. * Vamos a ver mi visualización favorita como ejemplo * T: Los datos se quedan en el DOM, son más estables allí según Mike Bostock.

class Line extends React.Component {

    static propTypes = {...}

    componentDidMount() {
        // D3 Code to create the chart
        this._chart = D3Line.create(
            this._rootNode,
            this.props.data,
            this.props.config
        );
    }

    componentDidUpdate() {
        // D3 Code to update the chart
        D3Line.update(
           this._rootNode,
           this.props.data,
           this.props.config,
           this._chart
        );
    }

    componentWillUnmount() {
        D3Line.destroy(this._rootNode);
    }

    _setRef(componentNode) {
        this._rootNode = componentNode;
    }

    render() {
        <div className="line-container" ref={this._setRef.bind(this)} />
    }
}

React para el DOM


class Line extends React.Component {

    static propTypes = {...}

    drawLine() {
        let xScale = d3.scaleTime()
            .domain(d3.extent(this.props.data, ({date}) => date));
            .rangeRound([0, this.props.width]);

        let yScale = d3.scaleLinear()
            .domain(d3.extent(this.props.data, ({value}) => value))
            .rangeRound([this.props.height, 0]);

        let line = d3.line()
            .x((d) => xScale(d.date))
            .y((d) => yScale(d.value));

        return (
            <path
                className="line"
                d={line(this.props.data)}
            />
        );
    }

    render() {
        <svg
           className="line-container"
           width={this.props.width}
           height={this.props.height}
        >
           {this.drawLine()}
        </svg>
    }
}
